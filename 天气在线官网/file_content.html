农历、节气、生肖

查询任何一个公历日期对应的农历日期、农历传统节假日、二十四节气、天干地支纪年纪月纪日、及生肖属相。查询任何一个公历日期对应的农历日期、农历传统节假日、二十四节气、天干地支纪年纪月纪日、及生肖属相。

请求地址示例
  • •从今天开始7天的农历和节气等

    https://api.seniverse.com/v3/life/chinese_calendar.json?key=your_api_key&start=0&days=7

  • •从指定日期开始7天的农历和节气等

    https://api.seniverse.com/v3/life/chinese_calendar.json?key=your_api_key&start=2014/1/31&days=7

参数
参数名 类型 默认值 必须 备注
key String 无 是 API密钥
strat String 0 否 开始时间
end String 0 否 结束时间
